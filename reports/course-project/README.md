# Course Project Report

## Files

- `report.tex` - основной исходник отчёта
- `report.pdf` - собранный PDF

## Build

Из корня репозитория:

```bash
tectonic reports/course-project/report.tex
```

PDF будет создан рядом с исходником.

## Notes

- титульные данные в `report.tex` нужно заменить на реальные ФИО и группу
- содержание отчёта написано по состоянию проекта на 1 апреля 2026 года
- базовые supporting-материалы лежат в `docs/` и `knowledge-base/obsidian/`
